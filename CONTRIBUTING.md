# 贡献指南

感谢您对 LuneOS for OnePlus 6 项目的关注！我们欢迎各种形式的贡献。

## 如何贡献

### 报告问题
- 使用 [Issues](https://github.com/[你的用户名]/luneos-oneplus6-lxc/issues) 页面报告问题
- 提供详细的错误描述和复现步骤
- 包括设备信息和系统版本

### 提交代码
1. Fork 本仓库
2. 创建功能分支 (`git checkout -b feature/amazing-feature`)
3. 提交更改 (`git commit -m 'Add some amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 创建 Pull Request

### 文档改进
- 改进文档和注释
- 添加使用示例
- 翻译文档

## 开发环境设置

参考 [BUILD.md](BUILD.md) 设置开发环境。

## 代码规范

- 使用有意义的提交信息
- 遵循现有的代码风格
- 添加必要的注释
- 测试您的更改

## 讨论与交流

- 使用 Issues 进行技术讨论
- 欢迎提出新功能和改进建议

## 许可证

贡献即表示您同意您的代码将根据 [GPL-3.0](LICENSE) 许可证发布。
