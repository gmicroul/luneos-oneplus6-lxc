# LuneOS for OnePlus 6 with LXC Support

为 OnePlus 6 (enchilada) 定制的 LuneOS 系统，包含完整的 LXC 容器支持。

## 特性

- ✅ 基于 LuneOS 最新版本
- ✅ 完整的 LXC 容器支持
- ✅ 一加6硬件完全驱动
- ✅ Docker替代方案（更适合移动设备）
- ✅ 开源构建，可自定义

## 硬件要求

- OnePlus 6 (enchilada)
- 解锁的 bootloader
- 至少 2GB 可用存储空间
- USB数据线
